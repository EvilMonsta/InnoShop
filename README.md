## Переменные окружения

SqlStrCon - Строка подключения к PostgreSQL 
ПРИМЕР "SqlStrCon": "Host=localhost;Port=5432;Database=DB;Username=postgres;Password=1234"

JwtSigningKey - JWT ключ
ПРИМЕР "JwtSigningKey": "mega-krutoy-slojnyy-kluch-kak-je-ego-uznat"

ProductsInternal__BaseUrl: url для продуктов
ПРИМЕР "ProductsInternal__BaseUrl": "https://localhost:7172"

InternalApiKey: внутренний ключ api
ПРИМЕР InternalApiKey: "dev-internal"
